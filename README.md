# SSRF Lab

A hands-on, multi-level training lab designed to demonstrate Server-Side Request Forgery (SSRF) vulnerabilities in a controlled local environment.

This project focuses on building practical intuition for identifying, understanding, and analyzing SSRF flaws — from basic cases to more advanced scenarios involving internal services and chained vulnerabilities.

## Key Features
- Progressive learning path with multiple levels
- Realistic vulnerable patterns inspired by real-world applications
- Safe, isolated environment for experimentation
- Designed for both learning and demonstrating security knowledge

## Levels

- **Level 1 — Basic SSRF**  
  Introduction to server-side requests and user-controlled input.

- **Level 2 — Validation Mistakes**  
  Bypassing naive input validation mechanisms.

- **Level 3 — Blind SSRF**  
  Identifying SSRF when no direct response is returned.

- **Level 4 — Internal Service Interaction**  
  Exploring access to internal resources and service flows.

- **Level 5 — Advanced SSRF Chaining**  
  Demonstrating how SSRF can lead to more complex impact scenarios, such as Remote Code Execution (RCE).
